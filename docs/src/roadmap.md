# Roadmap

TerminalOS development is organized into phases. Phases 1–14 are complete.

## Completed phases

1. **Terminal UI** — multi-pane Ratatui layout, tabs, keyboard focus
2. **Terminal Emulator** — PTY shells, ANSI, copy/paste, search, scrollback, split panes
3. **AI Chat** — multi-provider streaming chat with history
4. **Coding Agent** — slash commands with sandboxed file ops
5. **Git Assistant** — commit, diff, staging, blame, health checks
6. **Workspace Manager** — session persistence and restore
7. **Semantic Search** — tree-sitter chunks, embeddings, hybrid ranking
8. **Plugin SDK** — dynamic loading and marketplace
9. **Configuration** — profiles, themes, keybindings
10. **Developer Experience** — xtask, CI, snapshots, benchmarks, mdBook
11. **Groq & Onboarding** — first-class Groq provider, setup wizard
12. **Provider Switcher** — in-app hot-swap and chat error UX
13. **Split Panes** — horizontal/vertical splits per tab
14. **Release & Docs** — GitHub Releases and GitHub Pages deploy

## Future work

- In-terminal selection search polish
- Expanded plugin ecosystem
- Performance profiling and startup optimization
- Split layout persistence across workspace restore

See [ROADMAP.md](https://github.com/panditdhamdhere/TerminalOS/blob/main/docs/ROADMAP.md) in the repository for the canonical checklist.
