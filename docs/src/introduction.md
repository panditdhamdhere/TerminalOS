# Introduction

**TerminalOS** is a production-grade, cross-platform terminal application built in Rust. It combines a modern terminal emulator, AI assistant, coding agent, Git assistant, workspace manager, and developer dashboard into a single fast, keyboard-driven TUI.

## What makes TerminalOS different

- **Real shells** — PTY-backed bash/zsh with streaming output and ANSI colors
- **AI-native workflow** — chat, coding agent slash commands, and Git assistant in one UI
- **Workspace memory** — restore tabs, layout, branches, and environment variables across sessions
- **Semantic search** — tree-sitter chunking with hybrid keyword + vector ranking
- **Extensible** — dynamic Rust plugins with a bundled marketplace catalog
- **Configurable** — profiles, theme presets, and customizable keybindings
